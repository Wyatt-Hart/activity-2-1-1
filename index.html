    • Which tag is used for longer quotes?

      the <blockquote> tag is used to make longer quotes, and is usually indented by the browser

    • Where does the citation go when quoting a long text?

      The citation is put inside the <blockquote> tag (ex. <blockquote cite="wikipedia.com">)

    • Which tag is used for abbreviations?

      The <abbr> tag is used for abbreviations with a title attribute (ex. <abbr title="Science, Technology, Engineering, and, Math">STEM</abbr>)

      The BDO tag changes the direction text is written
